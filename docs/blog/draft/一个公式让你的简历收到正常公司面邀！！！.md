# 一个公式让你的简历收到正常公司面邀！

本条视频不要985，不要211，一本院校、二本院校、双非的同学都值得好好看看。

我们来看一下大家的简历通常会暴露什么样的问题。在最近改简历的过程当中，我发现这个现象越来越严重。

假设你投递的是正常公司、普通公司，也就是0到20人左右的初创公司除外。稍微注意一下：初创公司和某些集团的初创业务组是两个概念，要看它有没有靠山。如果有靠山的20人团队，那是可以进的；如果没有靠山的20人，有能力咱们就先不要进，因为确实有点不太稳定。

好，假设你是一本二本，只要是正常本科，也投了普通的正常公司，社保这些都是正常交的情况下，但是你的简历石沉大海，那大概率犯了以下两个问题，有可能两个都占：

## 常见问题

第一个就是**跨专业跨岗投递**，但是连一段跟岗位相关的经历都没有。不只是说校外，你哪怕校内都没有相关经历，那很抱歉，你准备挂了。

第二种情况就是你的专业院校经历都匹配，可是你的简历依然石沉大海。那我敢用绝对词说，你的简历一定出现了这个问题：**用词非常假大空，而且你把自己应有的能力都没有说出来**。

当你的简历和定位一旦出现问题，你一定做过的事情和你拥有的能力都说不清楚，这种情况下，你的整个秋招、春招、暑期实习等各种找工作都会出问题。

你买了个花里胡哨的简历模板，结果写出来令我瞠目结舌。今天解决的问题就是如何教你——我会用一条公式，针对性解决你的问题。

记住：是一个公式，通用解决一个针对性问题。

## 核心公式：把话说清楚

接下来视频在重点部分我都会强调，第一个你只需要学会一个技能：**把话说清楚**。我强调三遍：把话说清楚，把话说清楚，把话说清楚。

我先说这个公式，你听好了：

> **你在什么样的场景下 → 收集了什么样的信息 → 用这个信息做了一个什么样的判断 → 用这个判断又去做了一个什么样的动作 → 最后收获了一个什么样的结果**

很多同学会说，这听起来很像STAR法则，甚至有些接触过项目管理知识的同学会说这很像其他模型，我都学过。但是我告诉你，这个公式非常好用。

我不希望大家在实践过程中，还通过传统学习方式，带着某某理论去做事情。可能那个东西方便好记，但我觉得大白话最好记，也就是把场景过程中发生的事情拆出来，就是上面这个公式。

## 反面例子

我拿自己来举例，讲一下如果我犯你们那些错误，写简历会写成什么样。

假设我想向面试官表达，我具备优化策略的能力。如果我在网上学了一点碎片知识，我就会在简历前端写"我具备什么什么优化能力"，然后打一个冒号，说了很长一段话，结果什么信息都抓不住。

而且通常犯经典错误的人，大概率会这样写："我做了市场调研"，如果又学了一点碎片化知识，就会写成：

> 我做了300份的用户调研，针对三类人群的核心痛点进行入手，最终提升了转化率60%。

你觉得这样改真的很好，对吗？好，我从HR和业务方的视角来看这份简历，我会很想问你几个问题：

1. 三百份调研很多吗？你这300份的数据来源是线下还是线上？是发了朋友圈还是去街上调研的？资源不同，含金量完全不同。

2. 你说你根据用户调研分析出了痛点，请问你这个调研问卷里设置了什么内容就能抓住他们的痛点呢？这个痛点是你觉得的痛点，还是真正的数据反馈给你说明就是痛点？

3. 既然你说找到了痛点，假设我认为你前面说的话都是真的，你为什么不告诉我你分析了什么痛点呢？你甚至不肯告诉我，针对价格敏感型用户、针对追求品质的用户，我们分别做了什么样的调整，最终实现了转化率提升。

4. 你告诉我转化率提升了百分之几十，那我给你举一个极端的例子：你总共这一次真正来的消费者就只有三个人，你说提升到60%，那1.8个人里的0.8个人是怎么算出来的？

**与人方便就是与自己方便**。你把话讲清楚，让拿到简历的人能够顺畅地记住你更多的信息。他方便了，你就会被留下，你的简历被留下，你就会获得面试机会。

## 为什么要这么写？

主要是三个原因：

### 第一个原因：适用人群

假设你的这个岗位的学历要求和大致经历是匹配的。这条视频把话说清楚的技巧是通用的，但不能确保对你一定有用，两种情况除外：

- 第一种情况：你是985/211去投递BAT等顶级互联网梯队，这种方法有用，但如果没有三段实习垂直经历，大概还是会被筛下来。
- 第二种情况：你的院校低于本科以下，那大部分有些公司有一些内部潜规则，会直接把你筛掉。

今天这个技巧最终非常见效，且能提升简历通过率的人群，就是我说的**正常本科、普通本科，一本二本双非**。在这种情况下，你投递一个普通正常公司，这份简历通过第一关、收到面邀的概率绝对会大大提升。

因为把话说清楚这个能力，绝对能够帮你在这个批次的竞争者当中，甩掉70%的人。更何况你是校招生，社招有些人到现在都还是说不清楚话。

不过要说明：社招比校招本身就更要求岗位相关性，校招更多是看可迁移的能力，甚至是能力的潜质和潜力。

### 第二个原因：与人方便就是与自己方便

谁能够给拿到你简历的人留下印象，你就能留下来。

像我自己很忙，我昨天休假还在面试——我上秒在跟朋友吃饭，下一秒就端电脑出去面了一个小时。我对这个面试者很感兴趣，就是因为他的简历和他的开场白，包括他介绍自己的条理性，留住了我的注意力，留住了他的个人特点。于是我继续面他，准备给他通过。

### 第三个原因：专业HR看简历只有两个阶段

专业HR拿到简历，时间只会花在两个阶段：

1. 拿到简历的头5秒
2. 往下阅读了2到3分钟

大部分人简历5秒就被筛出去了。如果你的简历5秒能过关，你会获得黄金的2到3分钟。结果HR看到你巨长的教育背景，没有有效信息，写了一大堆主修课程，终于来到了相关经历，你承诺了你具有市场分析能力，结果犯了我刚刚举的例子一样的错误——经不起推敲。那我肯定大概率会把你筛出去。
