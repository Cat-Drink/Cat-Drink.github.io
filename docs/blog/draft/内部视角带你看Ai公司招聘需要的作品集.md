# 内部视角带你看AI公司招聘需要的作品集

前两天发了个AI公司招人，收到那大几十封简历里面啊，真的有一些被候选人做出来的个人网页给惊喜到了。

哈喽大家，我是博2。

今天说一下，一个个人作品集网页，在AI公司求职会是一个大大的加分项，而且要注意千万不能是那种AI流水线出来的状态哈。

今天呢就跟大家来分享一下，我们在看你的个人网页的时候，到底想看一些什么东西。

## 基础信息

首先最基础的就是基本信息和工作经验，这就不赘述了，因为这是最简单最快速能够知道你过往的一些信息。

然后我想跟大家强调的就是下面这两个点。

## 重点一：针对性展示工作产出

第一个就是工作部分，可以基于岗位来展示你过往的产出。

而且不仅仅像之前那样只展示一些项目或者数据成果。更多的是我们希望可以看到，比如说你做了什么工具来帮自己提效，或者说你的GitHub仓库里面会有哪些有意思的东西，这些都可以给到我们。

同时你也可以展示一下，你如何以AI native的方式去工作，这个工作流本身也非常加分。

## 重点二：审美和想象力

第二个点就是你的审美和想象力部分，这就是为什么我推荐做个人网页的原因。因为你的整个审美和态度就已经长在你的网页里面了。

比如说你的首页是怎么做的，整体的叙事排布是什么样的，我就可以非常清楚地看到你的思维逻辑和创造力，这个就非常有趣。

比如说像做运营的同学，如果你给到我的作品集是一个充满AI味的蓝紫色网页，我真的会觉得有一点点小减分。

这也是为什么我希望大家能够花一点时间坐在电脑前，好好把这个个人网页做出来的一个原因。

## 用AI快速搭建

其实现在对于小白用户真的非常非常友好，而且审美也能达到设计师级别。比如说我这次用AI搓的个人网页，它的前面hero效果是一个Mac显示进入终端的界面，这个动效一开始真的超出了我的预期，还有一些细节动画都非常厉害。

很多时候如果你不知道怎么去表述，或者你已经有一个视觉方案，你就直接把图片或者视频发给AI，它就可以帮你一比一把你的想法给复刻出来，变成代码。真的非常非常简单。

而且它很好的一点是可以一键部署，你直接把链接发给我们，我们就可以看到你的作品集了，省非常非常多事情。你也可以把更多的时间放在打磨你想要展示的内容上。
