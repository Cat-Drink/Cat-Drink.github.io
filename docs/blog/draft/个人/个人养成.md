# “脑雾”还是“脑腐”？如何解决？

### 1. 触发场景
- **脑腐**：刷完手机一两个小时之后，整个人瘫掉的感觉
- **脑雾**：生过病、熬过夜、长期压力后，做什么事情都慢半拍、反应慢

### 2. 炎症事件史
如果你在新冠后、流感过后、化疗过后，或者有慢性自身免疫病，**脑雾**的可能性会高得多。

### 3. 休息能否缓解
- **脑腐**：停下来不玩手机，一天左右就能有一定缓解
- **脑雾**：再怎么休息，一周可能都没什么改善

## 解决方法：机制不同，方法相反

### 脑腐的解决方法
核心是**重新训练你的奖赏系统**，让它能在低频度的活动里面也能感觉到满足。可以参考"七天多巴胺斋戒"的方法。

### 脑雾的解决方法
最主要的是**降低神经炎症**：
- 规律的有氧运动
- 地中海饮食
- 保证深度睡眠（深度睡眠期间淋巴系统会清理代谢废物）
- 必要时让医生帮你排查潜在的炎症原
