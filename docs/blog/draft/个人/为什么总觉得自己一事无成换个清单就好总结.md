# 为什么你总觉得自己一事无成？换个清单就好

一个很小但非常管用的建议：**把 To-do List 换成 Done List**。

## 问题在哪

很多人列待办清单（To-do List），但执行力不太好的时候，一天下来会觉得：

> 我今天什么事都没完成，我真是一事无成。

然后开始自我批评，越骂越没动力。

## 解决方法：Done List（已完成清单）

不要只写今天要做什么，**把今天你已经做完的事情写到一个 Done List 里**。

不管事情多小，做了就写下来：
- 按时洗澡 ✓
- 去找邻居办完了那件小事 ✓
- 整理了桌面 ✓
- 看完了一章书 ✓
- 出去走了 30 分钟 ✓

## 为什么管用

这是在**物理上给你的大脑制造闭环**：

- 每完成一件小事，就记录下来，人为给自己创造一点多巴胺
- 一天下来，你会发现「哇，我其实干了很多小事」，而不是「我什么都没干」
- 这些小闭环积累起来，就会变成更大的事

## 核心思路

不管做完一件多么小的事，都记录下来。慢慢你就会发现，你其实一直在前进，并不是一事无成。

这个方法特别简单，但非常管用，推荐试试。
