# 上大学到底有没有用？王浩宇的观点

> 上大学你要说有用也有用，你要说没用那也是真没用。

## 课堂知识可能过时，但你学到学习能力

机械坐在课堂里听的知识，有可能都已经过时了。**但你能学到的是「学习的能力」，而且学习能力是可以迁移的。**

举例：打游戏。不管玩什么新游戏，愿意去钻研、去看教学视频、去研究细节，就能慢慢变强。这种「遇到新事物快速上手学会」的能力，就是大学练出来的。

## 大学是人生第一次完全为自己负责

大学真正的价值不在于知识本身：

- 九年义务教育、高中，都是家长在管着你
- 到了大学，**没家长管了，这是你人生第一次完全凭借自己要去做成一件事情**
- 这是你第一次为自己负责

## 把大学读漂亮，人生会充满自信

当你靠自己把大学读得漂亮的时候，你完成了第一件完全自主做成的大事。

这种成功经验会给你自信——**接下来你做什么事都会有自信**。

## 总结

如果你正在读大学，一定要听这句话：**把大学读漂亮**。这本身就是最大的收获。
