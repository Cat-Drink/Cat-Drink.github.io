# 面试开头3分钟，面试官其实已经知道结果了

## 开场第一个问题暴露真相

但凡面试官开场第一个问题是「做个自我介绍吧」，**大概率说明他进会议室之前，根本没看过你的简历**。

## 自我介绍之后，两种提问方向两种结果

### 方向一：围绕你讲过的内容深挖 = 有戏 → 恭喜你

自我介绍完，如果面试官围绕你刚讲过的内容提问：
> 你刚刚提到在XX项目做了XX事，你当时具体是怎么做的？为什么要这么做？遇到了什么问题？

说明你的自我介绍**引起了共鸣**。

这其实是你埋的钩子生效了——把你的案例伏笔放在自我介绍里，就等着面试官来问。你准备好就能拿到高分。

### 方向二：不问经历，直接问「职业规划」「离职原因」 = 十有八九凉了

如果你的自我介绍说完，面试官不围绕你刚讲的对口点深挖，反而直接问：
- 你的职业规划是什么？
- 你为什么离职？

那这个时候，**你的面试十之八九一定是铁挂的**。

## 为什么会这样？

面试官快速过了一遍你的简历之后，他发现**顾虑大于肯定**。常见情况：

1. **公司背景不匹配**：你原来公司背景和现在这家公司完全搭不上
2. **岗位不匹配**：你之前工作岗位的性质，和这个职位出入很大
3. **简历太跳**：职业规划不清晰，对口工作年限很短
4. **你讲的偏题了**：你强调的关键点，面试官觉得不是核心

所以他根本没兴趣深挖你的项目经历，直接开始考察你的软因素，其实心里已经基本Pass你了。

## 总结

面试开头3分钟，走向已经很明显：
- ✅ 围绕你自我介绍的项目经历深挖 → 有戏
- ❌ 跳过经历直接问职业规划/离职原因 → 多半凉了

提前准备好自我介绍里的「钩子」，引导面试官问你准备好的问题，能大大提高通过率。
