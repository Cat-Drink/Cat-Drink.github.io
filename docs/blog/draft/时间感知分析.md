# 为什么感觉一年比一年过得快，是因为年龄增长的原因吗？

## 核心原因：重复

大脑要保存太多的记忆，所以重复的事情会直接打包压缩掉。你每天刷牙不可能一年记365次不同的细节，会直接打包成刷牙×365。

所以经常重复的事情约等于没有发生。只有不重复的事情才会让你对时间有感觉。

## 为什么小孩子觉得时间过得慢？

因为每天都要上学学新东西，不重复。不重复就无法压缩。

你每天上班重复做同样的事情，你生命当然过得快。

## 一个有趣的实验

不信你现在下班完别做平时做的事，随便找个没看过的但能看懂的课本仔细过脑子看一晚上，保证你觉得这个晚上过得巨慢无比。因为这个晚上对你的大脑来说是不可压缩的。
