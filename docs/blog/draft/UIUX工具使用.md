# UIUX Pro Max 网页开发工具使用指南

[UI/UX Pro Max - AI-Powered Design Intelligence](https://uupm.cc/)

UIUX Pro Max 是专门为 AI 编程打造的可搜索设计数据库，为 Web 开发提供设计智能支持。

## UIUX Pro Max 核心特点  

- 集成 57 种 UI 风格、95 种配色方案及多种字体配对

- 支持新旧基础站及多种图标类型

- 可包装为 Skill 集成到 AI Code 工具中，无需额外下载插件，配置后即可直接调用

## UIUX Pro Max 安装步骤  

1. **安装 Node.js**：需先下载 Node.js，可通过官网教程自行配置，或让 AI 协助下载

1. **安装插件到开发环境**：在终端输入命令安装

1. **验证安装**：在开发环境的 “skills” 中查看，若显示 UIUX Pro Max 相关文件（如 csv、py 文件）则安装成功

## UIUX Pro Max 使用方法  

- **调用方式**：在 solo 模式下，输入需求时指定风格（如 “生成 3D 风格网站”），AI 会自动调用 UIUX Pro Max 技能

- **风格参考**：可在 GitHub 搜索 “UIUX Pro Max skill”，通过中文翻译查看详细风格说明，生成时需明确指定风格类型

## 效果对比与示例  

- **优势**：相比直接使用 AI 生成，UIUX Pro Max 能显著提升前端页面美观度，减少 “AI 味”

- **示例**：使用像素卡通风格可快速生成用户登录、管理员登录、注册登录等页面，包含基础交互功能（如编辑资料），生成 3 个页面仅需约 5 分钟

- **自定义调整**：可通过补充具体需求提示词，对风格、交互细节进行微调

## 适用场景  

特别适合使用国产或性能一般的 AI 大模型时，全面提升前端设计效果，实现 “AI 设计大脑全面升级”
