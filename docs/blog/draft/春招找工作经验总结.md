# 春招找工作经验总结

## Boss直聘使用技巧

1. **识别招聘方职级**：如果boss上面，对方用的是人名，而不是"招聘专员""HR"，说明对方职级比较高，有话语权，亲自来boss上选人，沟通时多表现自己。

2. **优化搜索策略**：
   - 设置里面找到隐藏简历，对boss隐藏求职者偏好，对boss隐藏标签信息
   - 设置里面找到个性化推荐设置，把个性化职位推荐和个性化内容推荐关闭（个性化推荐关了以后骗子公司一下就没了）
   - 在消息界面刷新岗位
   - 搜索地区加专精特新企业
   - 尽量使用搜索搜索岗位

3. **刷新岗位技巧**：
   - 改变所在位置（同一个城市换一个区）
   - 改变学历（研究生改成本科，本科改成大专）：在在线简历的教育经历里把学历改低，因为boss上很多岗位只要求大专生，你就会刷出来一大堆新岗位
   - 改求职意向、多点新发布的职位、多换几个时间刷

4. **简历投递策略**：
   - 可以直接发简历的图片给Boss
   - 在boss或者自己所在地区搜索找工作，然后觉得各方面都OK的话，直接打电话过去问，投简历人家一般不会看或者已读不回，打电话最直接
   - 可以去百度查找公司电话

## 公司筛选方法

1. **看地址**：居民楼的不去，写字楼里贴满小广告的也不能去。

2. **看招聘信息**：啥玩意都是95后00后，气氛活跃，天花乱坠一大堆，就是不写福利待遇，那就是没有。

3. **看HR态度**：HR回答问题不耐烦，说明她工作不顺心，她都工作烦躁了，能是啥好公司。

4. **看人员结构**：
   - 都是95后00后说明干几年就跑路或者被优化了，没保障
   - 年轻人中年人多了都不好：年轻人多估计是留不住人牛马公司招牛马，中老年多了氛围压抑，还会不动声色的欺负新人
   - 要问部门岗位和年龄结构

5. **其他细节**：公司墙上或宣传视频里面逢年过节发个破粽子都要大合影的，赶紧撤。

## 面试准备建议

1. **面试前询问要点**：
   - 先问工作内容（可以直接了解到这个岗位是不是一人多岗，直接pass一人多岗的）
   - 然后问工作时间（有些是倒班但是岗位介绍里不明说，一定要问清楚还有单双休这些，单休直接pass）
   - 最后问一下五险一金（现在应该是都有五险了，小公司估计都是按最低基数，一金最好是有但是现在很多公司也不满足）
   - 问一下薪资结构（一般底薪+绩效的都会涉及到销售或有业绩压力）
   - 以上问题问完之后，也可以观察出hr是否专业负责（态度不好的直接pass）

2. **面试准备**：
   - 面试把自我介绍准备好，带两份简历和笔，找不到地方直接在软件问hr就好

## 薪资与工作平衡

- 满足早八晚五，双休，事情不多，不加班的岗位工资可能注定是不高的

## 经验年限策略

- 经验年限可以适当夸大：本人两年经验上岸一个金融大厂，部门里面最年轻。因为这个岗位需要经验年限规定。问我就说我5年，因为我那边业务量大，我两年的经验相当于你5年，可能还不止。面试官当场麻住。考虑也一下觉得也未尝没有道理。

- 对于硬性要求的灵活处理：做亚马逊运营，说要英语四级，没有，后面试通过了，需要四级，老实说没有，但是还是让入职了。如果当初说没有四级，可能连面试的机会都没有。
