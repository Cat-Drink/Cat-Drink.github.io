# 求职面试BOSS直聘实操经验整理
## 一、面试心态&面试渠道沟通技巧
1. **面试心态**
没事就找面试官聊天、多参与面试，多面之后通过率会变高；很多人初次进面试容易紧张、怕失败、怯于开口，多积累面试经验能缓解害怕新环境、陌生工作内容的焦虑。
有人靠大厂外包内推投递简历，面试官知晓简历情况后不会过多为难，面试体验更好。

2. **HR沟通话术与心态**
- 部分HR偏爱一句话概括自身情况，反感一问一答式低效沟通；沟通时简洁总结自身情况更容易获得好感。
- HR主动提及「我们最近遇到的挑战」属于利好信号，代表对方把你视作潜在合适伙伴。
- 区分优质HR：BOSS直聘上及时更新岗位状态（标注招满/暂停）的HR，责任心更强、招聘流程规范，长期合作稳定性更高；只机械回复10+客套话术、不认真沟通的面试官可以少投入精力。

3. **BOSS直聘投递巧思**
- 投递形式：可直接发送简历图片给BOSS，比纯文字打招呼更容易获得回复；
- 小号投递法：注册不登录的小号浏览筛选公司，筛选出心仪企业后再用主号投递，避免大号浏览痕迹被HR看到、频繁推送不匹配岗位；
- 大厂投递策略：死磕大厂难度高，可同步筛选50-200人规模中型公司，中型公司一面常能对接老板，实操面试机会更多；通勤远的岗位可直接筛选地铁沿线岗位，提升面试效率；
- 筛选避雷：若岗位方向和自身需求完全不匹配，可主动表现低兴趣，引导HR主动筛掉不合适岗位，节省自身时间。

## 二、主动联系老板直聊面试技巧
1. 操作步骤：
在BOSS直聘复制目标公司信息，查工商找到老板电话，直接致电；简单自我介绍、突出自身优势。
2. 对话场景示例：
老板短暂沉默无回应时，主动邀约线下见面；有求职者用该方式谈下底薪6000、周末双休、早九晚五、全国出差带旅游福利的岗位。

## 三、求职前期避坑须知（面试、租房、合同）
1. 不要主动频繁联系公司；
2. 不要聊几句就仓促答应前去面试；
3. 转正前租房规划：试用期可短租、住青旅；租房优先选择押一付一房源；
4. 试用期必须提前问清全部关键问题：
    - 转正薪资、工资架构、有无无责底薪；
    - 提成/奖金发放形式、试用期薪资标准、试用期时长；
    - 转正考核标准、每月社保扣费规则；
    - 节假日休息制度、加班补贴、上下班时间；
    - 劳动合同签约年限、离职相关约束条款。

## 四、其他求职心得
1. 大环境行情：当年求职难度偏高，但依旧是寻找工作机遇的一年；
2. 面试分层：金牌面试官、普通面试官体验差距大，不用执着于某一家公司的面试结果；
3. 成果反馈：坚持针对性投递、沟通筛选，最终有机会成功拿到offer。
