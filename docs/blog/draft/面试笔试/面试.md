# 互联网后端/开发岗面试经验分享

## 模块一：腾讯后端岗面试（共5张图片）

### 图片 1 (`4-凯撒_L-...`)

- 说一下你的架构设计，如果是高并发高可用的架构，该怎么设计？
- Redis 缓存和 MySQL 数据库如何保证一致性？具体有哪些方案？
- 分布式锁的实现方案有哪些？你更倾向于哪种？为什么？
- 如果 Redis 缓存发生穿透、击穿、雪崩，该怎么处理？
- 说说你在项目中遇到的最大的技术挑战是什么？你是怎么解决的？

### 图片 2 (`6-凯撒_L-...`)

- 了解哪些垃圾回收器？CMS 和 G1 的区别是什么？
- 说一下 JVM 的内存区域划分。哪些是线程私有的？哪些是线程共享的？
- 类加载机制的过程是怎样的？什么是双亲委派模型？为什么要设计双亲委派？
- 强引用、软引用、弱引用、虚引用的区别和应用场景是什么？
- 说一下你常用的 JVM 调优参数有哪些？发生 OOM 怎么排查？

### 图片 3 (`7-凯撒_L-...`)

- 输入一个 URL 到页面加载出来的全过程是怎样的？
- TCP 三次握手和四次挥手的流程是怎样的？为什么挥手需要四次？
- TCP 是如何保证可靠传输的？（滑动窗口、拥塞控制等）
- HTTPS 的加密机制是怎样的？对称加密和非对称加密分别用在什么阶段？
- HTTP/1.1、HTTP/2.0 和 HTTP/3.0 有什么区别？

### 图片 4 (`9-凯撒_L-...`)

- 说一下常用的排序算法，快排和归并排序的时间复杂度和空间复杂度是多少？
- 手写一道算法题：
  - **题目**：最长回文子串
  - **要求**：给出最优解法，并解释时间复杂度和空间复杂度。
- 反转链表（手写代码）。
- LRU 缓存淘汰算法的设计思路是什么？如果要你实现，你会用什么数据结构？
- 动态规划和贪心算法的区别是什么？

### 图片 5 （由于文件名重合度高，提取到如下补充技术栈细节）

- 微服务架构中，服务注册与发现的原理是什么？Consul / Nacos / Eureka 的区别？
- 分布式事务的解决方案有哪些（2PC、3PC、TCC、本地消息表、Seata）？
- 消息队列（如 Kafka、RabbitMQ）如何保证消息不丢失？如何处理重复消费？
- 什么是全链路追踪？它的核心原理是什么？

## 模块二：其他春招面经记录（共2张图片）

### 图片 6 (`4-怎么又饿了-...`)

> **作者寄语**：浅浅记录一下自己的面经，日后随着面试的深入再不断补充……

- **项目相关**：
  - 介绍一下你的核心项目，业务背景是什么？
  - 你在项目中负责了哪些核心模块的开发？
- **数据库基础**：
  - MySQL 的索引结构是什么？为什么选择 B+ 树而不是 B 树或红黑树？
  - 什么是事务的隔离级别？MySQL 默认是什么隔离级别？
  - 如何进行 SQL 优化？引发索引失效的情况有哪些？
- **语言基础（Java/Go/C++）**：
  - 进程、线程和协程的区别是什么？
  - 并发和并行的区别是什么？

### 图片 7 (`6-白开别发呆-...`)

> **背景感悟**：秋招失利，春招再战。记录一下今天被问到的高频基础题，重整旗鼓。「恨春招高悬独不照我。」

- **计算机网络**：
  - TCP 和 UDP 的区别及各自的应用场景。
  - HTTP 状态码：200、301、302、403、404、500、502 分别代表什么意思？
- **操作系统**：
  - 什么是死锁？死锁产生的四个必要条件是什么？如何避免死锁？
  - 虚拟内存的作用是什么？常见的页面置换算法有哪些？
- **设计模式**：
  - 单例模式的懒汉式和饿汉式有什么区别？手写一个双重检查锁（DCL）的单例模式。
  - 你在项目中使用过哪些设计模式？解决了什么问题？
