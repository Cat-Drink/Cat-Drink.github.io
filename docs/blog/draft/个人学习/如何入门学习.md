# 如何用 3 小时入门任意编程语言

Hello，兄弟们好啊，这里是 919 巷子。

今天这期视频，我想分享一个统一的思路，让你用 3 个小时的时间入门任意的编程语言。我认为，只要你有小学三年级的水平——英语、数学和逻辑，你就可以上手任何编程语言。注意，我这里说的是上手，而不是精通。你当了 10 年、20 年程序员，都不敢说自己精通任何一门编程语言。但是上手学会基本语法，我想是可以在非常短的时间内做到的。

我认为，与其一上来就关注特性，对于新手来说，倒不如一上来先关注共性。有没有哪些特点是学习所有编程语言都必须要面对、都逃不开的？我想是有的。我是科班出身，所以说我入门过非常非常多的语言，什么 C 啊、Java 啊、C# 啊、Go 啊、TS 啊，很多我都写过。我总结了九个入门步骤（Nine Steps for Beginners），接下来我来分享一下。

## 第一步：理解编程语言的本质

这是我从一个曾经的大厂 CTO 嘴里得知的，他说：任何编程语言本质上都是在操控硬件资源，在硬件资源之间去做平衡。硬件资源主要有五种：CPU、内存、磁盘、网络带宽，现在可能还多一个 GPU。这一步是为了让你对编程语言、对高级语言建立一个正确的、稍微有一点深刻的初步认知。

## 第二步：Hello World 与输入输出思维

所有编程语言你都会学 Hello World，对吧？但是你有没有想过为什么要学？这是一种程序的思维方式，叫做**输入输出思维**。程序本质上是需要跟人交互的，那人通过什么去交互呢？其中一个办法就是通过打印到那个黑黑的框框里面——那个叫做控制台，这是一种输出的方式。后面你可能会知道，可以通过网络交互，有 HTTP 协议、gRPC 协议，有 GUI 图形化的交互方式，有很多。但是最基础的就是我们说的"打印到控制台"。

## 第三步：基本数据类型

其实基本数据类型在不同语言里的定义是不一样的，每种语言的基本数据类型都不一样，而且这种不一样的背后都能体现那门语言自己的艺术，或者自己对于编程的思考。但以一个笼统的方式来理解编程世界，我们认为基本数据类型其实就四种：整数、浮点数（就是小数）、字符串、布尔。什么是布尔？就是真或假，True or false。

## 第四步：条件语句与循环语句

条件语句和循环语句，也就是 if 和 for。其实编程有三要素：你掌握了**顺序、条件、循环**之后，理论上就能编写任何复杂的程序了。包括现在的深度学习、神经网络、大模型，它的底层本质上都是由一个一个的顺序、条件和循环组成的程序，没有任何不同。举个例子，GPU 理论上是并行计算，它可能迅速运算一些矩阵，那你说我写好几个 for 循环去运算，最终得到的结果是一样的。

通常现在的语言都有传统的 for 循环（循环变量初始化、循环条件、循环变量的改变、循环体），还有那种人类更容易理解的 for-range 循环——直接迭代索引和对应的值。你看，你把不同的语言比对到一起学习，就会发现编程语言在表象上的设计是有很多共性的。

## 第五步：容器（数据结构）

你会发现绝大多数语言都有两种数据结构：一种叫**可变长度列表**，一种叫 **KV 表**（Key-Value 映射表），一个键对应一个值。学数据结构、学容器最关键的就是把增删改查的语法都看一下。怎么遍历一个 KV 表？怎么删除一个 Map 里面的 Key？遍历是有序的吗？这些最常见、最基本的问题，你是要理解的。

如果你不走后端开发的面试、不需要靠编程来吃饭的话，你可以不理解它的底层原理。但比如说遍历一个 Map，遍历是不是有序的，这种事情会直接影响你编程的结果——你一定要去看一下。

## 第六步：函数

函数代表了一种编程思维：**黑箱思维**，或者说封装的思维。我只关心输入和输出。除非硬件出故障，否则你运行 `function add(a, b)` 一万次、一百万次、一百万亿次，它得到的结果都是稳定的。你不用关心中间是什么逻辑。如果中间是一个非常复杂的逻辑，你也不用关心。你只知道输入什么、输出什么就行了，这是函数最重要的概念。函数也分内置函数和自定义函数，不同语言组织函数的方式也不同。

## 第七步：面向对象

面向对象是一种**组织代码的思维方式**。它把现实世界的东西抽象成代码。比如你开了一扇门，门这个物体就可以抽象为一个对象（Object），它可能有一些属性（长、宽、高）和一些方法（开、关）。不同语言实现面向对象的方式不一样，Go 用 struct 加方法，Python 和 Java 用 class，但它们背后的思想是相通的。

面向对象也有它的问题，当继承层级太深时会导致代码难以维护。所以又衍生出了"组合优于继承"的思想——Go 就是非常推崇组合的语言。不管用什么方式，核心目的是**更好地抽象现实世界、更好地组织代码**。你理解了面向对象，后面再学泛型也好、注解/装饰器也好，都只是为这个核心目的服务的语法糖。

## 第八步：依赖管理

不同语言的依赖管理方式不一样。比如 Java 用 Maven/Gradle，Go 有 go mod，Python 有 pip/poetry/conda。你需要知道怎么导入一个外部库（别人写好的代码），以及导入之后怎么在代码里面调用它。

## 第九步：上手写项目

当你把前面的概念都过完了，就可以上手写项目了。写一个能用的程序——比如一个 To-Do List、一个计算器、或者调用别人 API 的一个小程序。在这个过程中你会发现，其实所有语言在你刚入门的时候要做的事情是一样的，都是**操控硬件资源去处理数据**。如果你真正理解了这句话，并且去横向对比几门语言的基本语法如何做这件事，你就会发现：编程语言不过如此。

## 我为什么推荐 Python 作为第一门语言

我推荐 Python，有几个原因：

第一，Python 的语法足够简单、足够贴近自然语言。它的一些很复杂的代码——就是很长的代码——可能用几个很风骚的写法就搞定了，某种程度上你能体会到"编程语言的本质是操控硬件资源"。而且它对小白非常友好。

第二，Python 的生态比较丰富。它有很多第三方库——就是很多的"轮子"。你想造一台汽车，不需要自己造轮子，别人已经给你写好了，拿来用就行。比如 LangChain、PyTorch、scikit-learn、Flask，还有爬虫、测试脚本、画图的库，应有尽有。

第三，据我了解，很多非科班的同学本科有一门课就是 Python，那你可能要期末考试，对吧？

第四，如果你真的把编程思维深入骨髓，你会发现它真的能给你的生活带来便利。如果你不从事程序员工作，但生活或工作中有一些需要重复去做的事情，如果你能写脚本，把重复的事情交给机器去做——你可能看着电视、干着别的，后台脚本跑着，你的工作就完成了。提高效率、解放双手、节省时间。再加上现在有了大模型，编程的门槛比以前低了不知道多少。你如果会一些 Python 基础语法，就能看懂 AI 大模型给你输出的程序，然后根据自己的想法去调整，那不是一件很好的事吗？

## 推荐：夜曲编程 APP

说到 Python，我这里特别推荐零基础的同学去看一下"夜曲编程"这个 APP，它有六个非常好的优点：

1. **在线运行，无需配置环境**。纯零基础小白入门编程的第一座大山就是配环境、装环境，很麻烦。这个 APP 无需装环境，进去就有一个代码练习器，你直接线上编程，立刻能输出结果。

2. **概念解释细致生动**。可以说是不厌其烦——每一个非常细的点，比如什么叫变量，在我看来已经非常好理解的事，可能对于一个编程小白来说就是一个问题，它把你可能遇到的问题全部替你想好了。

3. **关键概念卡片化整理**。每个关键概念都以卡片形式收集起来，专门有一个地方可以回顾。你当前学了哪些概念、哪些是重要的，能不能用自己的话把概念说出来。如果忘了，可以专门去一个地方看，非常形象、简单、直接。

4. **注重基础巩固**。它怕你忘了，学了基本概念后会让你做几个选择题，引导你主动思考。

5. **在线题库丰富**。题目的数量有一两百个，让编程小白也能体验在线做题的快感。题目有难度说明，判题响应时间也足够快，挺方便的。

6. **在线学习导师**。它已经这么掰开了揉碎了讲，你还有问题的话，还可以问老师，不是吗？
