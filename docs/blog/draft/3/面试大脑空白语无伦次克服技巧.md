# 面试大脑空白语无伦次？试试这些克服技巧

一到面试环节瞬间紧张，脑子全空白了，私下准备得很充分，但是开口就语无伦次。相信很多人都遇到过这种情况，这里分享几个实用的调整技巧：

## 1. 调整心态：把面试当成平等交流，而非被考核

紧张的根源在于：你太想完整表达自己了，而且把面试官放在一个高位。

试着调整心态：**把面试当成平等交流，而不是被考核审视**。当你把位置放平了，紧张感自然会减少很多。

## 2. 做减法：用短句框架，放慢语速

给自己做减法：
- 把高频问题的答案写成**短句**，默念熟悉框架即可
- 语速**放慢半拍**，给自己缓冲思考的时间

我见过很多厉害的人，他们说话都是很稳的那种慢。不追求完美回答，**条理清晰就很加分**。

## 3. 换视角：把每次面试都当经验积累

把每次面试都当做积累经验的过程，你不仅是在找工作，也是在锻炼自己的面试能力。这样就算失败了，也不会感到沮丧，因为你确实收获了经验。
