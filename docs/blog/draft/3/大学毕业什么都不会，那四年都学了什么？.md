# 大学毕业什么都不会，那四年都学了什么？

关于这个问题，我有两个核心观点：

## 观点一：其实无论学什么专业，本科四年确实不太能学到什么东西

很可能你学的这个专业，在本科四年阶段，你确实没有学到什么能直接用的专业技能。毕业的时候感觉自己什么都不会，这种感受非常正常。

## 观点二：无论你学什么专业，都不会白学

虽然专业技能可能没学到多少，但是由于你在这个领域当中学习了这么长时间，**它在潜移默化中培养了你在这个领域思考问题、看待问题的方式**。

这些思维方式的积累，对于你的未来——无论你是考研跨专业考，还是以后从事与本专业不相关的工作——其实都是有用的。

## 亲身经历：我的电子信息工程

我当年高考选的是电子信息工程专业，这个专业是我爸妈帮我选的。我爸妈农民出身，他们并不了解大学是什么样，也不了解电子信息工程到底是干什么的，只不过是听别人说"这个专业近几年很吃香，学了好找工作"，在我们当事人都不了解的情况下，我就浑浑噩噩读了这么一个专业。

事实上，我这四年学得不是很好，总的来说就是没学好。结果就是：那么好就业的一个专业，我作为这个专业出身的，大学毕业的时候还是找不到工作。

后来我考研跨专业考，再后来读研读博都是跨专业，现在我的工作也跟当年的电子信息没有什么关系了。

**可是**，曾经那四年理工科的学习过程当中，积累的一些看待问题、思考问题的方式方法，一直持续到今天，我都能说它是有用的。我虽然没有学明白什么专业知识、专业技能，但是这段学习经历，我觉得直到今天都是有用的。

甚至有些敏感的学生在听我讲考研政治课时，都能反馈说："老师，我感觉你跟别的政治老师最大的不同之处，就是你在用理工科的思维去讲考研政治的知识。"

所以这都是能感受得到的。

## 总结

大学四年，你可能在专业技能上学不到太多东西，但这段学习过程并不是完全没有用。**它给你留下的思维方式和看待问题的视角，会一直伴随你，无论你将来走到哪个领域，都是有用的。**

因此我说：学不到显性的技能，但不代表这段经历白瞎了——那些潜移默化中塑造的思维方式，才是大学四年真正给你的礼物。
