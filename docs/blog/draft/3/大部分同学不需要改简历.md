# 大部分同学不需要改简历

## 简历的三个阶段

### 阶段一：格式期
- 简历格式已经定型，业内已有统一标准
- **做法**：跟着标准简历模板改就行——上面有的写，上面没有的别写
- 不要在第一阶段浪费时间（如写目标求职城市、目标薪资等）
- 格式再完美，如果不对口，仍然是垃圾简历

### 阶段二：对口期
- 你要投什么岗位，你会做什么岗位的活，面试官能一眼看出来
- 没有实习经历的同学、天坑专业（生化环材、土木、语言教育等）转行的同学最容易出问题
- **简历不对口就是垃圾，再怎么改都没用**
- 方法：去找 20 个相同的目标岗位，搞清楚这个岗位要什么样的人、要求什么能力

### 阶段三：人设期（高阶技巧）
- 直接影响面试官问问题的走向
- 适用于大厂、外企、产品经理等要求较高的岗位
- **让面试官能从简历看出你的成长路径，人为控制面试问题走向**

## 核心提醒

> 简历对口只是基础要求。不要偷懒，搞清楚目标岗位需要什么能力，才是真正的求职基础课。
