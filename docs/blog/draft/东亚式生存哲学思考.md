# 东亚式生存哲学思考

## 作者：忘丘山

### 存在主义悖论：为了避免死，拒绝活

我们能算清的只有一件事：不开始，就不会结束。

这是唯一的确定性。也是最大的悲剧。

有个哲学家叫海德格尔，他说人是"向死而生的"——正因为会死，活着才有意义。

爱情也是。正因为会分手，在一起才珍贵。正因为会痛，快乐才真实。

但我们东亚青年太聪明了，聪明到发现了一个漏洞：如果不开始，就不会分手；如果不活，就不会死。

我们选了一条最安全的路。然后发现：这条路上，什么都没有。

### 东亚式风险规避文化

东亚小孩从小被教育：想清楚再行动，别冒险。

"万一考不上怎么办？""万一摔着怎么办？""万一被人骗了怎么办？"

这些"万一"像一道道篱笆，把我们圈在安全的区域里。我们学会了评估风险，学会了趋利避害，学会了在行动之前先想好退路。

### 东亚逻辑中的损失认知

后来我才明白：在东亚逻辑里，没发生的事，不算损失。你错过的，就当不存在。

这是一种很聪明的活法。但也是一种很遗憾的活法。

### 东亚式的聪明

你避开了所有痛，也避开了所有好。

很聪明的逻辑。东亚式的聪明。
