# MySQL8.0安装配置(Linux)

## 1. 下载

```bash
wget https://dev.mysql.com/get/Downloads/MySQL-8.0/mysql-8.0.21-linux-glibc2.12-x86_64.tar.xz
```

## 2. 解压

```bash
tar xvJf mysql-8.0.21-linux-glibc2.12-x86_64.tar.xz
```

## 3. 重命名文件夹

```bash
mv mysql-8.0.21-linux-glibc2.12-x86_64 mysql-8.0
```

## 4. 创建data文件夹

```bash
mkdir data
```

## 5. 创建用户组以及用户和密码

```
groupadd mysql 
useradd -g mysql mysql
```



## 6. 授权用户

```bash
chown -R mysql.mysql /usr/local/mysql-8.0
```

## 7. 初始化

```bash
./mysqld --user=mysql --basedir=/usr/local/mysql-8.0 --datadir=/usr/local/mysql-8.0/data/ --lower_case_table_names=1 --initialize
```

> **备注**：红框位置为初始密码，一定要复制粘贴记住！

## 8. 编辑my.cnf文件

```bash
vim /etc/my.cnf
```

文件内容参考：

```ini
[mysqld]
basedir=/usr/local/mysql-8.0
datadir=/usr/local/mysql-8.0/data
socket=/tmp/mysql.sock
log-error=/usr/local/mysql-8.0/data/mysql.err
pid-file=/usr/local/mysql-8.0/data/mysql.pid
character-set-server=UTF8MB4
symbolic-links=0
lower_case_table_names=1
```

## 9. 修改mysql.server文件

```bash
vim support-files/mysql.server
```

查找 `Starting MySQL` 字眼，在如下位置增加 `--user=root`

## 10. 添加mysqld服务到系统

```bash
cp -a ./support-files/mysql.server /etc/init.d/mysql
```

## 11. 授权以及添加服务

```bash
chmod +x /etc/init.d/mysql
chkconfig --add mysql
```

## 12. 启动MySQL

```bash
service mysql start
```

## 13. 查询启动状态

```bash
service mysql status
```

## 14. 将mysql命令添加到服务

```bash
ln -s /usr/local/mysql-8.0/bin/mysql /usr/bin
```

## 15. 登录数据库，修改密码

```bash
mysql -uroot -p
# 输入步骤7的密码
```

## 16. 修改root密码

```sql
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123456';
flush privileges;
```

## 17. 修改访问权限

```sql
use mysql;
update user set Host='%' where User='root';
flush privileges;
```
