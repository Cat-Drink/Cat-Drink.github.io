# 广州美食推荐

No.1广州最好吃的早茶：陶居居
凤爪、豉汁排骨、艇仔粥、金钱肚、虾饺都好吃，水牛奶菠萝包给到夯爆了，内陷是流动的，奶味非常香，除了会喷出来没有看到缺点。

No.2广州最好吃的粤菜：小炳胜
黑松露脆皮文昌鸡很好吃，鸡皮脆脆的，肉质很嫩，皮和肉之间有汁水，蘸松露汤汁也比较入味，给到夯爆了；蜜汁黑叉烧稍微有点肥，但是反而产生一种爆汁的口感，吃多了可能会腻，点一例就够了！

No.3广州最好吃的糖水：陈苏苏 料巨多，你喜欢吃芒果，一定要点 多芒小丸子，芒果很多很清甜！鬼口水混冰淇淋白雪，必须给到一个夯！

No.4广州最好吃的火锅：潮发（潮汕牛肉火锅）
雪花肉，吊龙肉，匙仁肉，嫩牛肉，牛腩，三花趾、五花趾请全部都点！总体给到夯，牛肉可以根据标识的秒数烫，很嫩很美味，带筋的部分脆脆的口感非常棒，不蘸调料就很好吃，服务员帮忙调了蘸料和沙茶酱，蘸过也非常好吃。我之前点的鸳鸯锅，但是实测觉得清汤更适合，保留肉本来的味道，也方便蘸调料！
