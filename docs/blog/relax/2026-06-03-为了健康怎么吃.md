# 为了健康怎么吃

## 【多吃】
1. 维B含量高的蔬果：香蕉、菠菜、西兰花、生菜、葡萄、杨桃
2. 维C含量高的蔬果：菠萝、猕猴桃、橙子、草莓、木瓜、甘蓝
3. 富含Omega3食物：三文鱼、金枪鱼、鳕鱼、鲱鱼
4. 微量元素高的食物：海带、木耳、鱼肉、红枣、生蚝、牛肉

## 【少吃】
1. 高糖食物：冰激凌、蛋糕、老式糕点
2. 腌制食物：泡菜、腊肉、罐头、咸菜
3. 油炸食物：炸鸡、烧烤、薯条、炸串、油条等
4. 辛辣食物：辣椒、大葱、花椒、生姜、酒、胡椒等
