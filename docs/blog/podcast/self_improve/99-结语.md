# 结语

你觉得迷茫，不代表你软弱，不代表你没用，不代表你比那些"看起来很有方向感"的同学差。有时候那些看起来很有方向感的人，只是把那个方向感包装得比较好。或者，他们的方向是父母给的，不是自己找的——那种方向，稳定，但未必是他自己的。

而你在这里认真面对这个问题，已经是一个很重要的开始。

不用找到完美的答案。先找到一个够用的方向，往那里走一步，看看感觉，调整，再走。

清醒不是一种状态，是一种选择。它不是知道了所有答案，而是你开始用自己的眼睛去看路。

如果你愿意，今晚就可以做那个小的尝试：把手机放到另一个房间，早睡，明天早起的时候，给自己留十分钟时间，不带目的地坐一会儿。

看看会发生什么。

> *"你在三四月做的事，在八九月自有答案。"*
