# MaxWell

### Maxwell的底层实现原理

一个 MySQL 数据库的增量数据捕获（CDC, Change Data Capture）工具，它将自己伪装成一个slaver，利用主从复制原理，读取 MySQL 的 binlog（Binary Log）来捕获数据变化，并将这些变化实时地发送到如 Kafka、Kinesis、RabbitMQ 或其他输出端

#### 【项目】maxwell是怎么把mysql数据同步到kafka的？

启用MySQL服务器的Binlog，利用MySQL的主从复制原理，将maxwell伪装成slave，从master同步数据

#### 【项目】mysql的业务数据是通过maxwell同步到kafka的，那为什么不用flink cdc？

跟离线数仓保持一致
